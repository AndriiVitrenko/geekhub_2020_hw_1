Это файл с макетом. Макет разбит по слоям и сгрупирован по папкам:

  • number - нумерация блоков. (номера верстать не нужно)
  • text - текстовые блоки (шрифт Arial, размер 14 пикселей, межстрочное расстояние 22 пикселя)
